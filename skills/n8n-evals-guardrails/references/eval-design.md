# Eval Design

## Minimum viable evaluation pack

- 10 to 30 representative cases
- at least 3 failure-mode cases
- expected outputs or metrics
- one regression suite for each major workflow change

## Metric families

- correctness
- formatting
- grounding
- tool safety
- latency

## Guardrail placements

- before model input
- before tool execution
- before user-visible response
