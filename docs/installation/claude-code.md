# Claude Code Installation

## Install manually

1. Clone this repo.
2. Copy the contents of `skills/` into your Claude Code skills directory.
3. Keep this repo available locally if you want the references folders to remain linked in place.

Typical path:

- macOS/Linux: `~/.claude/skills/`
- Windows: `%USERPROFILE%\\.claude\\skills\\`

## Recommended routing

- install all skills, not a subset
- keep the root [AGENTS.md](../../AGENTS.md) nearby when working in repos that include n8n
- default to `n8n-mcp-toolsmith` before any n8n MCP use
