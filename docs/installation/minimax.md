# MiniMax Installation

MiniMax coding environments vary by host product, so this repo provides a portable instruction pack rather than a single install mechanism.

## Install

1. Keep the repo available locally or in your workspace.
2. Use `adapters/minimax/AGENTS.md` as the system or project-level instruction pack.
3. Reference canonical skills under `skills/` for deep workflow guidance.

## Best practice

- mention the exact skill names in your prompt
- keep validation and deployment skills paired with build skills
- treat the canonical `skills/` directory as the durable source of truth
